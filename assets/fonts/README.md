# Fonts

De huisstijlfonts in deze map zijn lokaal gekoppeld via `@font-face` in `assets/css/styles.css`.

- `Futura PT Heavy.ttf`: koppen (`font-weight: 700`)
- `Futura Light.otf`: lichte Futura-tekst (`font-weight: 300`)
- `Open Sans Light.ttf` en `Open Sans Light Italic.ttf`: `font-weight: 300`
- `Open Sans Regular.ttf` en `Open Sans Italic.ttf`: `font-weight: 400`
- `Open Sans Semibold.ttf` en `Open Sans Semibold Italic.ttf`: `font-weight: 600`
- `Open Sans Bold.ttf` en `Open Sans Bold Italic.ttf`: `font-weight: 700`
- `Open Sans Extrabold.ttf` en `Open Sans Extrabold Italic.ttf`: `font-weight: 800`

Alle fontkoppelingen gebruiken `font-display: swap`, zodat de tekst tijdens het laden zichtbaar blijft.
